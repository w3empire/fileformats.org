Extensions de fichier de développeur

Les types de fichiers pertinents pour la création de logiciels se trouvent dans la catégorie Developer Files. Cette section comprend les fichiers utilisés dans le développement d’un programme logiciel, tels que le code source du projet, les bibliothèques, les en-têtes et les classes. Sont également inclus les composants et les objets qui ont été compilés.
