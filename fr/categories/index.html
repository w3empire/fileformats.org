Diverses extensions de fichiers
Il existe différents types de fichiers dans la catégorie Fichiers divers qui n’appartiennent à aucune autre catégorie.
Extensions de fichier de développeur
Les types de fichiers pertinents pour la création de logiciels se trouvent dans la catégorie Developer Files.
Extensions de fichier de feuille de calcul
Les données sont conservées en lignes et en colonnes dans les cellules d’un fichier de feuille de calcul.
Extensions de fichier de mise en page
Les documents connus sous le nom de fichiers de mise en page peuvent contenir à la fois des informations de texte et d’image.
Extensions de fichier d'image disque
Dans un fichier image disque, vous trouverez une copie conforme d’un disque dur ou d’un autre support de stockage.
Extensions de fichiers audio
La catégorie connue sous le nom de ‘Fichiers audio’ contient des formats audio compressés et non compressés.
Extensions de fichier CAO
Les fichiers de conception assistée par ordinateur (CAO) peuvent inclure des modèles bidimensionnels ou tridimensionnels.
Extensions de fichier codées
Les fichiers qui utilisent une méthode d’encodage pour enregistrer leur contenu sont dits ’encodés'.
Extensions de fichiers compressés
L’économie d’espace sur votre disque dur est l’une des principales raisons pour lesquelles les fichiers compressés existent.
Extensions de fichier d'image de caméra
Les fichiers bruts d’image de l’appareil photo sont stockés dans un format brut qui conserve toutes les informations enregistrées par le capteur de l’appareil photo.
