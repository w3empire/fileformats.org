数据文件扩展名

所有计算机都使用数据文件来存储信息。 它们可以与软件一起预先打包，也可以由最终用户制作。 尽管确实存在基于文本的文件，但大多数数据文件使用二进制格式。 数据库、项目和保存的文档都是数据文件类型。

注意：数据文件部分包括过多的子部分。 任何不属于其他部分的文件都放在此处。
