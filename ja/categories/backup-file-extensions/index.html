バックアップ ファイルの拡張子

バックアップ プログラムに関連する個々のファイルとデータのバックアップ コピーは、どちらも [バックアップ ファイル] セクションに含まれています。 ソフトウェアは頻繁に新しいバックアップ ファイルを自動的に作成します。 すべての優れたバックアップ プログラムは、データの完全なコピーと増分コピーの両方を保存します。
