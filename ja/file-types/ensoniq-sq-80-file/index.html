「.EFS」ファイルとは何ですか?

EFS ファイル形式は、SQ-80 にオーディオ データを保存するために使用されます。これは、SQ-80 に固有の独自のファイル形式であり、他のデバイスやソフトウェアで開いたり再生したりすることはできません。EFS ファイルがあり、そこに含まれるオーディオ データにアクセスしたい場合は、SQ-80 と互換性があり、EFS ファイルを読み取ることができるデバイスまたはソフトウェアを使用する必要があります。

詳しくは

Ensoniq SQ-80 は、1987 年にリリースされたデジタル シンセサイザーおよびシーケンサーです。 SQ-80 は、強力なシンセサイザー エンジンと、ユーザーが独自の音楽を作成および録音できるシーケンサーを内蔵しています。電子ミュージシャンの間で人気があり、その多用途性と幅広いサウンドを生成する能力で知られていました。
