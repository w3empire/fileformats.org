「.UST」ファイルとは？

USTファイルは、VOCALOIDやVOCALOID2と同様のボーカルシンセサイザーであるUTAU（「Ust-Based Text-To-Speech」の略）ソフトウェアで使用されるファイルです。USTファイルには、歌声のピッチ、タイミング、およびその他の特性を合成および変更する方法に関する情報が含まれているため、ユーザーは独自の歌声を作成およびカスタマイズしたり、音楽プロジェクト用のボーカルトラックを作成したりできます.

UTAUでは、ユーザーはソフトウェアの組み込みエディターを使用してUSTファイルを作成および編集できます。このエディターでは、歌詞を入力し、合成された音声がどのように歌うかを指定できます。UST ファイルは、DAW (デジタル オーディオ ワークステーション) などの他のオーディオ ソフトウェアにインポートして、さらに編集や制作を行うこともできます。

詳しくは

UST ファイルは UTAU ソフトウェアで再生でき、他のオーディオ ソフトウェアで使用するために他の形式にエクスポートすることもできます。

UTAUでUSTファイルを再生するには、ソフトウェアでファイルを開いて再生ボタンを押すだけです。UTAUは、USTファイルに含まれる情報を使用して歌声を合成し、コンピューターのスピーカーまたはオーディオ出力デバイスから再生します。

UTAU では、UST ファイルを再生するだけでなく、MIDI (.MID) ファイルや VOCALOID (.VSQ) ファイルなど、他の形式にエクスポートすることもできます。UST ファイルを MIDI または VOCALOID 形式にエクスポートすると、これらの形式と互換性のある別のオーディオ プログラムで合成された歌声を使用する場合に便利です。

UST ファイルを WAV ファイルにレンダリングするオプションもあります。これは、ほとんどのデバイスやソフトウェアで再生できる一般的なオーディオ ファイル形式です。UST ファイルを WAV ファイルにレンダリングするには、UTAU の「レンダリング」または「エクスポート」オプションを使用するか、別のオーディオ ソフトウェアを使用して UST ファイルをインポートし、WAV ファイルにレンダリングします。
