Extensões de arquivo de layout de página

Documentos conhecidos como arquivos de layout de página podem conter informações de texto e imagem ao mesmo tempo. Além disso, eles contêm dados de formatação, que especificam as margens, o tamanho da página e como o conteúdo é disposto na página.

Para a produção de publicações imprimíveis como jornais, revistas e brochuras, os documentos de layout de página são frequentemente usados.
