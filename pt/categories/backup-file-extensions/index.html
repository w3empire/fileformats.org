Extensões de arquivo de backup

Cópias de backup de arquivos individuais e dados pertencentes a programas de backup estão incluídos na seção Arquivos de backup. O software frequentemente cria novos arquivos de backup por conta própria. Todos os bons programas de backup armazenam cópias completas e incrementais de seus dados.
