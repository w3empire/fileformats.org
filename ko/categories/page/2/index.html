데이터 파일 확장자
데이터베이스, 프로젝트 및 저장된 문서는 모두 데이터 파일 유형입니다.
데이터베이스 파일 확장자
데이터베이스 파일의 정보는 필드가 있는 테이블 형식 또는 테이블 형식으로 저장됩니다.
디스크 이미지 파일 확장자
디스크 이미지 파일에서 하드 드라이브 또는 기타 저장 매체의 사본을 찾을 수 있습니다.
래스터 이미지 파일 확장자
래스터 이미지는 전체 이미지를 나타내는 고유한 색상이 할당된 픽셀 격자 형태를 취합니다.
벡터 이미지 파일 확장자
벡터 이미지를 만들 때 픽셀 대신 경로가 사용됩니다. 이미지의 선과 기하학적 모양은 이러한 경로로 나타낼 수 있습니다.
비디오 파일 확장자
비디오 파일 범주에서 다양한 비디오 형식을 찾을 수 있으며, 각 형식은 비디오 정보를 인코딩 및 압축하기 위해 서로 다른 코덱을 사용합니다.
설정 파일 확장자
운영 체제 환경 설정 및 컴퓨터 프로그램 설정은 ‘설정’이라는 파일에 저장됩니다. 구성 파일은 기본 설정 파일이라고도 합니다.
스프레드시트 파일 확장자
데이터는 스프레드시트 파일의 셀 내의 행과 열에 보관됩니다.
시스템 파일 확장자
컴퓨터 운영 체제 파일(예: Mac OS, Windows 및 Linux에서 사용되는 파일)은 시스템 파일 범주에서 찾을 수 있습니다.
실행 파일 확장자
실행 파일에는 Windows 및 Mac 프로그램, 스크립트 및 매크로가 모두 포함됩니다.
