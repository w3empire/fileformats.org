DOC# 파일이란 무엇입니까?

LibreOffice는 OpenOffice.org의 소스 코드를 기반으로 하는 무료 사무용 생산성 제품군입니다. 저장된 문서(DOC 파일)를 편집할 때 임시 잠금 파일 DOC#을 생성하고 LibreOffice에서 문서를 닫으면 삭제합니다. 이 파일은 여러 사람이 동시에 사용할 수 없도록 문서를 잠그는 데 사용됩니다.

DOC# 파일 형식에 대한 추가 정보

LibreOffice에서 파일 작업을 할 때 작업 중인 다른 파일 형식에 대해서도 임시 잠금 파일이 생성됩니다. 예를 들어 .ODT 파일을 편집하면 “.odt#” 파일이 생성됩니다.

DOC# 파일로 저장된 문서는 “.~lock.MyDocument.doc#” 명명 규칙을 따릅니다. 여기서 “MyDocument"는 현재 편집 중인 파일의 이름입니다. DOC 파일을 편집할 때 동일한 폴더에 DOC# 파일도 생성됩니다.

확장자가 DOC#인 파일은 파일을 여는 사용자, 문서를 마지막으로 저장한 날짜와 시간, LibreOffice 잠금 정보가 포함된 디렉터리 경로(Windows에서는 [사용자])에 대한 정보가 포함된 일반 텍스트 문서입니다. \AppData\Roaming\LibreOffice3).
