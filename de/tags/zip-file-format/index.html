ZIP-Dateiformate

Die verlustfreie Datenkomprimierung wird durch das als ZIP bekannte Archivdateiformat unterstützt. Eine ZIP-Datei kann eine oder mehrere komprimierte Dateien oder Verzeichnisse enthalten. Obwohl DEFLATE der beliebteste Komprimierungsalgorithmus ist, der vom ZIP-Dateiformat unterstützt wird, gibt es andere Optionen.

Dieses Format wurde 1989 von Thom Henderson entwickelt und erstmals im PKZIP-Dienstprogramm von PKWARE, Inc. verwendet, um das frühere ARC-Komprimierungsformat zu ersetzen. Andere Software-Dienstprogramme als PKZIP begannen bald, das ZIP-Format zu unterstützen.
