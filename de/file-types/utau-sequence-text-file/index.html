AUFMACHEN

Was ist eine UST-Datei?

Eine UST-Datei ist eine Datei, die von der UTAU-Software (kurz für „Ust-Based Text-To-Speech“) verwendet wird, einem Vocal-Synthesizer ähnlich wie VOCALOID und VOCALOID2. UST-Dateien enthalten Informationen darüber, wie die Tonhöhe, das Timing und andere Eigenschaften einer Singstimme synthetisiert und modifiziert werden, sodass Benutzer ihre eigenen Singstimmen erstellen und anpassen oder Gesangsspuren für Musikprojekte erstellen können.

In UTAU können Benutzer UST-Dateien mit dem integrierten Editor der Software erstellen und bearbeiten, der es ihnen ermöglicht, Texte einzugeben und anzugeben, wie die synthetisierte Stimme sie singen soll. UST-Dateien können auch in andere Audiosoftware wie DAWs (Digital Audio Workstations) zur weiteren Bearbeitung und Produktion importiert werden.

Mehr Informationen

UST-Dateien können in der UTAU-Software wiedergegeben und auch in andere Formate zur Verwendung in anderer Audiosoftware exportiert werden.

Um eine UST-Datei in UTAU wiederzugeben, können Sie die Datei einfach in der Software öffnen und die Wiedergabetaste drücken. UTAU verwendet die in der UST-Datei enthaltenen Informationen, um die Singstimme zu synthetisieren und sie über die Lautsprecher oder das Audioausgabegerät Ihres Computers wiederzugeben.

Zusätzlich zur Wiedergabe von UST-Dateien ermöglicht UTAU Benutzern auch den Export in andere Formate, wie z. B. MIDI- (.MID) oder VOCALOID-Dateien (.VSQ). Das Exportieren einer UST-Datei in das MIDI- oder VOCALOID-Format kann nützlich sein, wenn Sie die synthetisierte Singstimme in einem anderen Audioprogramm verwenden möchten, das mit diesen Formaten kompatibel ist.

Sie haben die Möglichkeit, eine UST-Datei auch in eine WAV-Datei zu rendern, ein gängiges Audiodateiformat, das auf den meisten Geräten und Software wiedergegeben werden kann. Um eine UST-Datei in eine WAV-Datei zu rendern, können Sie die Option „Rendern“ oder „Exportieren“ in UTAU verwenden, oder Sie können eine andere Audiosoftware verwenden, um die UST-Datei zu importieren und in eine WAV-Datei zu rendern.
