Estensioni di file codificate

I file che utilizzano un metodo di codifica per salvare il loro contenuto sono detti “codificati”. In questa categoria sono inclusi archivi crittografati e non compressi, nonché file di testo con codifica binaria. Quando si trasferiscono i dati, è prassi comune crittografare i file per motivi di sicurezza e stabilità.
