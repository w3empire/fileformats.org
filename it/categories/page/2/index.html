Categorie

Estensioni dei file di dati
Database, progetti e documenti salvati sono tutti i tipi di file di dati.
Estensioni di file di eBook
Gli eBook, o file di libri elettronici, possono essere letti su eReader come Kindle, Nook, iPad e Sony Reader.
Estensioni dei file del foglio di calcolo
I dati vengono conservati in righe e colonne all’interno delle celle di un foglio di calcolo.
Estensioni dei file di gioco
I file relativi ai videogiochi, come i file di salvataggio, si trovano nella categoria File di gioco.
Estensioni dei file di immagine del disco
In un file immagine del disco, troverai una copia carbone di un disco rigido o di un altro supporto di archiviazione.
Estensioni dei file delle immagini della fotocamera
I file raw delle immagini della fotocamera vengono archiviati in un formato raw che conserva tutte le informazioni registrate dal sensore della fotocamera.
Estensioni dei file di immagini raster
Le immagini raster assumono la forma di una griglia di pixel, a ciascuno dei quali viene assegnato un colore univoco per rappresentare l’immagine complessiva.
Estensioni di file di immagini vettoriali
Quando si crea un’immagine vettoriale, vengono utilizzati percorsi anziché pixel. Linee e forme geometriche in un’immagine possono essere rappresentate da questi percorsi.
Estensioni dei file di layout di pagina
I documenti noti come file di layout di pagina possono contenere contemporaneamente informazioni di testo e immagini.
Estensioni dei file plug-in
I programmi esistenti possono beneficiare di funzionalità e funzionalità aggiuntive utilizzando i file plug-in.
