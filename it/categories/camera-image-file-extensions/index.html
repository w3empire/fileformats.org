Estensioni dei file delle immagini della fotocamera

Le fotocamere digitali producono file di immagine chiamati file raw della fotocamera. Sono memorizzati in un formato raw che conserva tutte le informazioni registrate dal sensore della fotocamera. Grazie all’ampia varietà di sensori per fotocamere digitali, è disponibile anche un’ampia varietà di formati raw per fotocamera.

Non è possibile aprire un file Camera Raw a meno che il software non sia compatibile sia con il formato del file che con il modello di fotocamera utilizzato per scattare la foto.
