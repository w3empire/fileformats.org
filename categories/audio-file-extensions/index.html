Audio File Extensions

The category known as “Audio Files” contains both compressed and uncompressed audio formats. These formats store waveform data that can be accessed and played back using software designed specifically for that purpose.

MIDI files, musical scores, and audio project files are all included in this category, despite the fact that these types of files do not typically store audio data.
