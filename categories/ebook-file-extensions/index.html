eBook File Extensions

eBooks, or electronic book files, can be read on eReaders like the Kindle, Nook, iPad, and Sony Reader. However, eReader software makes it possible to read them on a personal computer. Many eBook formats are compatible with a wide range of reading devices, but some proprietary eBook formats can only be read on very specific computers.
