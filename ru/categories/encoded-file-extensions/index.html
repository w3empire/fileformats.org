Закодированные расширения файлов

Файлы, которые используют метод кодирования для сохранения своего содержимого, называются «закодированными». В эту категорию входят как зашифрованные, так и несжатые архивы, а также текстовые файлы с двоичным кодированием. При передаче данных общепринятой практикой является шифрование файлов из соображений безопасности и стабильности.
