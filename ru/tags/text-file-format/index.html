Форматы текстовых файлов

Одна форма файла, которая может быть сохранена на компьютере, называется текстовым файлом и структурирована как набор строк текста. В файловой системе компьютера есть текстовый файл, который используется для хранения данных.

Текстовые файлы, сохраняемые в современных операционных системах, таких как Microsoft Windows и системах, подобных Unix, не содержат специального символа конца файла (EOF). Вместо этого файловые системы на этих платформах отслеживают размер файла в байтах.

Большинству текстовых файлов нужны разделители в конце строки, которые могут обрабатываться несколькими способами в зависимости от операционной системы. Разделители в конце строки необходимы для всех текстовых файлов.
