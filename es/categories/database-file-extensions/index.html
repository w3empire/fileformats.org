Extensiones de archivo de base de datos

La información en un archivo de base de datos se almacena en formato tabular o de tabla, con campos. A lo que nos referimos como ‘registros’ en el mundo de las bases de datos son las entradas individuales en sí mismas. Los sitios web dinámicos suelen utilizar bases de datos para almacenar la información que utilizan.
