Extensiones de archivo de desarrollador

Los tipos de archivos relevantes para la creación de software se pueden encontrar en la categoría Archivos de desarrollador. En esta sección se incluyen archivos utilizados en el desarrollo de un programa de software, como el código fuente del proyecto, bibliotecas, encabezados y clases. También se incluyen componentes y objetos que han sido compilados.
