Extensiones de archivos de fuentes

Los archivos de fuentes son archivos que contienen caracteres de escritura utilizados por el sistema operativo de la computadora y varios otros programas de computadora. Las computadoras Macintosh y Windows admiten los formatos OpenType y TrueType, que se utilizan para almacenar las fuentes más modernas.
