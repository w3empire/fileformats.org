Formatos de archivos binarios
Un archivo de computadora que no es textual a veces se denomina “archivo binario”. Algunos formatos de archivos binarios, como los primeros documentos de Microsoft Word, en realidad guardan tanto el texto del documento como su información de formato como datos binarios.

Formatos de archivo de texto
Una forma de archivo que se puede almacenar en una computadora se conoce como archivo de texto y está estructurado como una colección de líneas de texto. Dentro del sistema de archivos de una computadora, hay un archivo de texto que se utiliza para almacenar datos.
