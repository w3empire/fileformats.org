¿Qué es un archivo DOC#?

LibreOffice es una suite de productividad de oficina gratuita basada en el código fuente de OpenOffice.org. Crea un archivo de bloqueo temporal DOC# al editar un documento guardado (un archivo DOC) y lo elimina cuando el documento se cierra en LibreOffice. Este archivo se utiliza para bloquear el documento para que varias personas no puedan utilizarlo simultáneamente.

Más información sobre el formato de archivo DOC#

Cuando se trabaja en un archivo en LibreOffice, también se genera un archivo de bloqueo temporal para cualquier otro tipo de archivo en el que esté trabajando. La edición de un archivo .ODT, por ejemplo, da como resultado la creación de un archivo “.odt#”.

Los documentos guardados como archivos DOC# siguen la convención de nomenclatura “.~lock.MyDocument.doc#”, donde “MyDocument” es el nombre del archivo que se está editando actualmente. Cuando se edita un archivo DOC, también se genera un archivo DOC# en la misma carpeta.

Los archivos con la extensión DOC# son documentos de texto sin formato que contienen información sobre el usuario que abrió el archivo, la fecha y hora de la última vez que se guardó el documento y la ruta al directorio que contiene la información de bloqueo de LibreOffice (en Windows, este es [usuario] \AppData\Roaming\LibreOffice3).
